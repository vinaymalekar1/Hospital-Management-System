# Hospital-Management-system

This repository contains my Java project created using IntelliJ IDEA.  
I have also uploaded a ZIP file so anyone can download and run the project easily.

 *How to Run the Project:
1. Click the ZIP file in this repository.
2. Click Download.
3. Extract the ZIP file on your computer.
4. Open IntelliJ IDEA.
5. Click File → Open, then select the extracted project folder.
6. Open the Java file that contains `public static void main(String[] args)`.
7. Click the Run ▶ button in IntelliJ.

*Requirements:
Java JDK 8+
IntelliJ IDEA
